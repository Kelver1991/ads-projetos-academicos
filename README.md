# ADS Projetos Academicos

Repositorio com paginas desenvolvidas durante estudos de Analise e Desenvolvimento de Sistemas.

Este projeto funciona como registro de aprendizado em HTML e CSS, com estrutura basica de site pessoal/academico.

## Conteudo

- `index.html`: pagina inicial
- `sobre.html`: apresentacao pessoal
- `formacao.html`: formacao e estudos
- `portfolio.html`: projetos e atividades
- `contato.html`: pagina de contato
- `estilo.css`: estilos compartilhados

## Tecnologias

- HTML5
- CSS3

## Como abrir

Abra o arquivo `index.html` no navegador.
